# actor-analyser

Анализатор акторной активности - инструмент, предназначенный для исследования и отладки YDB.

## Целевой дизайн

Во время работы акторной системы каждый поток ведет короткий thread-local двоичный циклический лог, в который записываются записи об отправке и о начале обработки каждого сообщения акторной системы, а также о простое потока и некоторые дополнительные данные об отдельных акторах и сообщениях.
Запись крайне легковесная и оказывает минимальное влияние на производительность, что позволяет использовать ее в продакшн окружении для ретроспективного анализа проблем.
Запись лога может быть поставлена на паузу путем рассылки по древовидной иерархии нод кластера специального сообщения.
Лог может быть скачан с ноды и проанализирован при помощи Анализатора.
Анализатор позволяет просматривать все записанные и скачанные с нод логи в визуальном формате с различными скоростями в прямом и обратном направлении, а также фильтровать отображаемую активность по различным критериям, строить трассы.

## Основные отличия текущей версии от целевой

Логи в текстовом формате через cerr, требуют наложения патча, существенно замедляют работу системы, часть необходимой информации в логах отсутствует.

Команды остановки записи и соответствующей инфратруктуры нет.

Получить и склеить логи с разных нод сейчас невозможно.

Фильрации/поиска при отображении нет.

Построение трас очень сырое.
